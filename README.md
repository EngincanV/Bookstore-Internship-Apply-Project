## Installation
1. Clone the repository.
```sh
git clone https://github.com/EngincanV/Bookstore-Internship-Apply-Project.git
```
2. Install node modules via npm.
```sh
npm install
```
3. Run the app.
```sh
npm start or yarn start
```
